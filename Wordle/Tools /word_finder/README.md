### Das Python Script funktioniert wie folgt

* Man fügt eine grosse Wortliste ein mindestens 15, maximal 100’000 Wörter
* Das Python Script filtert die Wörter anhand der einstellungen im feld " Word length“.
* Wörter mit ä, ö, ü, anderen Sonderzeichen oder Zahlen werden ignoriert, da ich sie für dieses Projekt nicht benötige.
* Das Script zeigt nur die passenden Wörter an.
* Unten wird angezeigt, wie viele Wörter gefunden wurden.
*	Das Script läuft nur, wenn mindestens 15 Wörter (durch Leerzeichen getrennt) eingefügt wurden – sonst läuft es nicht.


### Warum dieses Tool:
* Es spart Zeit, ist übersichtlich und hilft bei der Vorbereitung von Wortlisten für Spiele oder Analysen


## Voraussetzungen

Um dieses Programm zu starten, wird Folgendes benötigt:

- Python 3 (von python.org)
- Tkinter (ist bereits in Python enthalten)
- Keine zusätzlichen Bibliotheken notwendig

Das Programm funktioniert auf macOS, Windows und Linux
